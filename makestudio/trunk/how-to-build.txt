Building is tested with Delphi XE5 and XE7

First synchronize SVN: https://optimeas.de/public/makestudio to your favorite local folder.
<makestuditrunk>=<your folder>\trunk

Settings for Namespaces (Tools-Options)
System;Xml;Data;Datasnap;Web;Soap;Winapi;Vcl;System.Win;Vcl.Imaging

A. Build from Source

  1. Open MakeStudioGroup.groupproj in ..\makestudio\trunk\source\framework
  2. Compile All
  3. Generate makestudio.dll assembly: 
      "<DOTNETBINPATH>\tlbimp" <makestuditrunk>\source\framework\makestudio.tlb /transform:dispret /verbose /namespace:makestudio
  4. Copy makestudio.dll to <makestuditrunk>\bin
  5. Run <makestuditrunk>\setup\makestudio.iss with Inno Setup 


B. Build with MakeStudio

  1. Install the latest MakeStudio version from 
    http://www.optimeas.de/de/service/downloads.html?view=document&id=29:makestudio-4-0&catid=109:makestudio 
  2. For XE5 Load <makestuditrunk>\setup\buildD19.jmk
  2. For XE7 Load <makestuditrunk>\setup\buildD21.jmk
  3. Run <makestuditrunk>\setup\build<xx>.jmk
  4. You will find <makestuditrunk>\setup\output
